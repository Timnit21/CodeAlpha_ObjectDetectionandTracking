# CodeAlpha-Object-Detection-Tracking

A professional-grade, real-time object detection and tracking system developed as part of my **CodeAlpha AI Internship**. This project utilizes state-of-the-art computer vision models to identify and track objects in real-time.

## 🎯 Project Overview
This system processes video input to detect and track specific objects using the **YOLOv8** framework. It is optimized for high-speed performance, featuring persistent object tracking and real-time visualization.

## 🚀 Key Features
* **Real-time Detection:** Powered by YOLOv8-nano for low-latency performance.
* **Persistent Tracking:** Maintains unique IDs for objects as they move across frames.
* **Performance Metrics:** On-screen FPS counter for real-time analysis.
* **Professional Logging:** Automatically records annotated video output to a local `.avi` file.
